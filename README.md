# smartmet-engine-osm

Part of [SmartMet Server](https://github.com/fmidev/smartmet-server). See the [SmartMet Server documentation](https://github.com/fmidev/smartmet-server) for a full overview of the ecosystem.

## Overview

The OSM engine provides access to OpenStreetMap geographic data for SmartMet Server. It enables plugins to query OSM data for use in map products and location-based services.
